# **App Name**: AdminView

## Core Features:

- User List: Display a list of users with their details, including ID, login, name, balance, deposits, withdrawals and winnings.
- Date Range Filtering: Allow filtering the displayed data by a selected date and time range.
- Data Export: Enable exporting report data.
- Search: Enable real-time search functionality based on user data attributes.
- User Management: Enable management of user deposits and withdrawals.
- Dark Mode Toggle: Allow toggling between light and dark UI modes for user preference.

## Style Guidelines:

- Primary color: Slate blue (#708090) for a calm, professional interface.
- Background color: Light gray (#F0F8FF) for light mode and dark charcoal (#36454F) for dark mode.
- Accent color: Teal (#008080) for interactive elements to maintain a visual consistency.
- Body and headline font: 'Inter', sans-serif, for a modern, readable look.
- Use minimalist, line-based icons for clarity and to reduce visual clutter.
- Use a grid-based layout to organize data, ensure readability, and improve information scanning speed.
- Subtle transition animations for dark mode and loading states to provide a smooth user experience.
